# Portfolio (static)

Simple static portfolio site. Open `index.html` in a browser or use a local static server.

Quick start (macOS):

```bash
# open in default browser
open index.html

# or run a simple static server (recommended for SPA features like client-side routing)
# if you have Python 3 installed:
python3 -m http.server 8000
# then open http://localhost:8000
```

Files added:
- `index.html` — main HTML
- `styles.css` — site styles
- `script.js` — small interactions
- `assets/avatar.svg` — placeholder avatar

Customize the content (name, projects, links) and replace the avatar with your image.
